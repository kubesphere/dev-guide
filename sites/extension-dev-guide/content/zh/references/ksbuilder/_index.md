## ksbuilder

ksbuilder 是 KubeSphere 扩展组件的命令行接口。

```bash
ksbuilder [flags]
```

### 可选项

```bash
  -h, --help   help for ksbuilder
```

## ksbuilder create

执行以下命令创建新的 KubeSphere 组件。

```bash
ksbuilder create [flags]
```

## ksbuilder package

执行以下命令打包组件。

```bash
ksbuilder package [flags]
```

## ksbuilder publish

执行以下命令将组件发布到扩展市场。

```bash
ksbuilder publish [flags]
```

## ksbuilder unpublish

执行以下命令将组件从扩展市场中移除。

```bash
ksbuilder unpublish [flags]
```

## ksbuilder version

执行以下命令查看组件版本。

```bash
ksbuilder version [flags]
```
